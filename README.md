<h1 align="center">Obol SDK Examples</h1>
    
## Description

The repo contains a JS+TS Examples for using [Obol-SDK](https://www.npmjs.com/@obolnetwork/obol-sdk).

## Installation And Running the examples

```bash

$ yarn

# JS
$ yarn run start

# TS
$ yarn run build
$ yarn run start

```
